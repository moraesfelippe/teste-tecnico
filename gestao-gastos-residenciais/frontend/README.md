# Front-end — Controle de Gastos Residenciais

Front-end em React + TypeScript (Vite) para o sistema de controle de
gastos residenciais. Veja o README na raiz do repositório para a visão
geral do projeto e instruções completas de execução (back-end + front-end).

## Comandos

```bash
npm install   # instala as dependências
npm run dev   # inicia em modo desenvolvimento (http://localhost:5173)
npm run build # gera a build de produção (verifica tipos + empacota)
npm run lint  # roda o linter (oxlint)
```

A API é esperada em `http://localhost:5199` (ver `src/services/api.ts`).
